# winebuilder-image
Repository to host images of my [wine-builder container on DockerHub](https://hub.docker.com/repository/docker/nellokudo/wine-builder), based on `steamrt` and adapted to be able to build Wine forks, from Vanilla Wine to Valve's.

You can find usage of those in my repos:
- [WineBuilder](https://github.com/NelloKudo/WineBuilder)
- [spritz-wine](https://github.com/NelloKudo/spritz-wine)
- [spritz-wine-aur](https://github.com/NelloKudo/spritz-wine-aur)
